Q learning 2D

Highly experimental attempt at integrating reinforcement learning: Q learning. The Q value of an action from some state is the average permanence of synapses activating the action from that state, minus the initial permanence value.

The action layer columns are interpreted to produce an action: 100 columns are allocated to each of the four directions of movement, and the direction with most active columns is used to move the agent.

Adjustments to the Q value, based on reward and expected future reward, are applied to the permanence of synapses which directly activated the action (columns). This adjustment applies in the action layer only, and it does this instead of the usual spatial pooling.

Exploration arises from boosting neglected columns, primarily in the action layer.

The reward is -3 on normal squares, -200 on hazard squares and +200 on the goal square. These are divided by 100 for comparison to Q values on the synaptic permanence scale.

This example is episodic: when the agent reaches either the goal or a hazard it is returned to the starting point.

The input is the location of the agent via coordinate encoder, plus the last movement as distal input.
